XV Международная цифровая олимпиада «Волга-IT»

Каждый айтишник найдет свою компанию!

Волга-IT — это международная цифровая олимпиада для школьников и студентов в возрасте от 16 до 23 лет. Наша уникальность — задания, основанные на реальных кейсах IT-компаний, с которыми им пришлось столкнуться. Попробуй себя в решении рабочих задач и найди компанию мечты!

Дисциплины

1C программирование

  • знание среды разработки «1С:Предприятие 8.3»
  • знание одной из нотаций для описания бизнес-процессов IDEF 0, IDEF 3, или EPC
  • умение структурировать и обрабатывать информацию, которую вам сообщит «пользователь» вашей будущей системы

Backend-разработка: Web API

  • хорошие знания одного из следующих языков: C#, Java, Python, Go
  • знакомство со Swagger, JSON Web Token (JWT)
  • умение разрабатывать RESTful API сервис
  • знания СУБД PostgreSQL

Web-дизайн

  • владение графическим редактором (Figma, Sketch, Adobe XD, Photoshop)
  • насмотренность и чувство стиля
  • корректный подбор цветов, символов, картинок, соответствующих стилю и задачам сайта
  • умение правильно расставлять акценты

Автоматизация тестирования (Java)

  • знание основных принципов тестирования
  • уверенное знание языка Java
  • навыки работы с Selenium, JUnit/TestNG, RestAssured
  • понимание принципов REST
  • умение работать с DevTools

Искусственный интеллект и анализ данных

  • умение работать с данными: реляционные таблицы, индексы, идентификаторы; csv, json, xml, текстовые файлы; SQL
  • основы программирования на языке высокого уровня (Python, C#, Java и т.д.), а также объектно-ориентированное программирование
  • знакомство с базовыми библиотеками работы с Python ( NumPy, Pandas, TensorFlow, Sci-Kit Learn, SciPy, PyTorch, Keras, Matplotlib и т.д.)
  • математическое моделирование, статистика и умение решать нестандартные задачи

Мобильные приложения Flutter

  • Знание основ мобильной разработки и опыт работы с кроссплатформенным фреймворком Flutter
  • Базовые навыки работы в VS code и консоли
  • Базовое умение создавать пользовательские интерфейсы и интегрировать функциональность в мобильные приложения

Телекоммуникации и информационная безопасность

  • знание сетевых служб и протоколов
  • знание оборудования для локально-вычислительных сетей, локальной и сетевой информационной безопасности и защиты информации
  • знание технологий: SSH, VPN, OpenSSL, Docker, iptables, DNS, DHCP, IPv4, IPv6, ARP, RBAC
